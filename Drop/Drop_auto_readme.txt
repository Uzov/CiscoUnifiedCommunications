      При при исходящих звонках с Cisco кодека на H323 терминалы/кодеки третьих производителей (Polycom, Yealink) на Expressway-E имеет место преобразование протокола H323 <-> SIP (Interworking). Сразу после установления соединения появляется нежелательная удалённая презентация в виде "чёрного окна" на Cisco кодеке. При входящих звонках на Cisco кодек имеет место та же проблема. Для её решения на всех Cisco кодеках установлены и запущены JavaScripts, которые при определённых условиях сбрасывают нежелательную презентацию. Текст скрипта автоматического сброса представлен в файле drop_auto.js.docx
